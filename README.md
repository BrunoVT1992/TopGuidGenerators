## What Is a GUID / UUID?

A **GUID** (Globally Unique Identifier) or **UUID** (Universally Unique Identifier) is a 128-bit value designed to be unique across space and time.

Example: 7e760907-099b-41d9-83f7-c290dc6f059a


They are commonly used for:

- Database primary keys
- Distributed systems
- Event sourcing
- Correlation IDs
- APIs
- Logs
- Anything that must *never collide*

Yes, the collision probability is *absurdly small*.
